# 📌 Repository Overview

This repository contains end-to-end Natural Language Processing (NLP) projects developed using Python and Machine Learning.

The projects demonstrate the complete NLP workflow, including data cleaning, text preprocessing, feature engineering, model training, hyperparameter tuning, model evaluation, and prediction. The **Amazon Review Sentiment Analysis** project is additionally deployed as an interactive web application using **Streamlit**.

The objective of this repository is to build production-ready NLP solutions while following industry-standard machine learning workflows and best coding practices.

# 🚀 Projects Included

# 1️⃣ Amazon Review Sentiment Analysis

## 📖 Project Overview

Developed an end-to-end sentiment analysis system capable of classifying Amazon product reviews into **Positive** and **Negative** sentiments.

The project follows a complete machine learning pipeline, beginning with text preprocessing and feature engineering, followed by model training, hyperparameter tuning, evaluation, and deployment through a Streamlit web application.

---

## ✨ Features Implemented

### ✔ Data Cleaning

- Removed HTML tags
- Removed punctuation
- Removed special characters
- Removed extra whitespaces
- Converted text to lowercase

### ✔ Text Preprocessing

- Tokenization
- Stopword Removal
- Lemmatization

### ✔ Feature Engineering

- Bag of Words (BoW)
- TF-IDF Vectorization

### ✔ Machine Learning

- Trained multiple classification models
- Compared model performances
- Selected the best-performing model
- Performed Hyperparameter Tuning

### ✔ Deployment

- Production-ready Python scripts
- Interactive Streamlit Web Application
- Real-time sentiment prediction using user input
